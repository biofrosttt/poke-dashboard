# PokéDex Dashboard

A Pokémon dashboard built with **React + TypeScript + Tailwind CSS + Vite**.

## Features

- Dark-first UI with atmospheric gradients
- Live search by name or Pokédex number
- Multi-type filter chips
- Infinite scroll (loads 40 at a time)
- Type-colored cards with glow effects
- Shiny sprite toggle (per card and in modal)
- Animated stat bars
- Evolution chain navigator (click to switch)
- Pokédex flavor text + genus
- Floating sprite animation
- Abilities tab with hidden ability badges
- Moves list tab

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:5173](http://localhost:5173)

## Stack

- React 18 + TypeScript
- Tailwind CSS v3
- Vite 5
- Axios
- PokéAPI (https://pokeapi.co)
